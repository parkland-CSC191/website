Welcome to CSC 191

Comprehensive coverage of SQL (Structured Query Language) including data retrieval and manipulation, sorts, joins, sub-queries, built-in functions, constraints, objects, transactions and granting and revoking privileges, utilizing both class time and online time for labs, hands-on training and group collaboration. A semester-long progressive class project will focus on building a viable web application utilizing the skills learned in this class.

The Structure of this class

This class is labeled as a "hybrid" class, where we spend almost two hours "in class" and another chunk of time "outside" of class (ie, online). In class you can expect that I will spend a little bit of time unwrapping the topic of the day, but most of the time I hope to have you "doing" SQL. The "online" portion of class will be a project that you work on slowly, week by week, either by yourself or in a small group, that builds until completion by the end of the semester. In between those two things, I expect students to read the book and develop one or more questions based on the material (each week).
